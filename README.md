Overview

This website is designed to automatically generate and print a PDF file after a user fills out a leave application form. The system simplifies the leave filing process by reducing manual paperwork, ensuring accurate data entry, and producing a ready-to-print PDF document for official use.


---

Benefits

Faster leave filing process

Paperless form filling

Professional and consistent document format

Convenient and time-saving


---

Developers

Developed as part of an academic/project requirement to demonstrate automation, document generation, and user-centered system design.
